## 🚀 Quick Start

1. **Clone & Install**

   ```bash
   git clone https://github.com/yourusername/nike-brand-page.git
   cd nike-brand-page
   npm install
   ```

2. **Run Development Server**

   ```bash
   npm run dev
   ```

3. **Open Browser**
   Navigate to `http://localhost:5173`
